Password Security Tips

Use Strong Passwords

Create passwords with at least 12 characters, mixing uppercase, lowercase, numbers, and symbols.

Avoid Common Patterns

Don't use dictionary words, personal information, or predictable patterns like "123456" or "password".

Use Unique Passwords

Never reuse passwords across multiple accounts. Each account should have its own unique password.

Enable 2FA

Add an extra layer of security with two-factor authentication whenever possible.
